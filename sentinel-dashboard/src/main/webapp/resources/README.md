# Sentinel-Admin

## 环境要求

- Node.js > 6.x

## 编码规范

- HTML/CSS 遵循 Bootstrap 编码规范，参考：https://codeguide.bootcss.com/
- JavaScript 遵循 Airbnb JavaScript Style Guide，参考：https://github.com/airbnb/javascript/tree/es5-deprecated/es5

## 开发流程

1、安装依赖

```
npm i
```

2、开发环境构建

```
npm start
```

3、生产环境构建

```
npm run build
```
